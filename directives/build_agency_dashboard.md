# 지사 대시보드 구축 지침 (build_agency_dashboard.md)

## 목표
플러스 지사용 대시보드 페이지(`pages/agency/dashboard.html`)를 구축한다.

## MVP 1차 스코프
- 로그인한 지사의 기본 정보 표시
- 긴급 요청 등록 바로가기
- 내 요청 목록 (상태별: 모집중/매칭완료/종료)
- 현재 플랜 + 남은 요청 횟수 표시

## 페이지 구성
| 섹션 | 내용 |
|------|------|
| 상단 헤더 | 로고 + 내비게이션 + 로그아웃 |
| 웰컴 배너 | "안녕하세요, {지사명}" + 현재 플랜 배지 |
| 통계 카드 | 이번달 요청 수 / 매칭 성공 / 남은 요청 횟수 |
| 요청 목록 | 내 긴급 요청 목록 (상태 필터) |
| 빠른 액션 | 새 요청 등록, 요청 목록 전체 보기 |

## 디자인
- 다크 테마 유지
- 상태 배지: 모집중(레드), 매칭완료(그린), 종료(그레이)

## 완료 조건
- HTML 문법 오류 없음
- 반응형 (375px ~ 1440px)
