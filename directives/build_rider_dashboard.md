# 라이더 대시보드 구축 지침 (build_rider_dashboard.md)

## 목표
용병 라이더용 대시보드 페이지(`pages/rider/dashboard.html`)를 구축한다.

## MVP 1차 스코프
- 로그인한 라이더의 기본 정보 표시
- 오늘의 출동 가능 여부 토글
- 최근 지원한 요청 목록 (미니 카드)
- 빠른 링크: 요청 목록 조회, 내 프로필

## 페이지 구성
| 섹션 | 내용 |
|------|------|
| 상단 헤더 | 로고 + 내비게이션 + 로그아웃 |
| 웰컴 배너 | "안녕하세요, {이름}님" + 출동 가능 토글 |
| 통계 카드 | 총 출동 횟수 / 이번달 수익 / 평점 |
| 요청 목록 | 최근 긴급 요청 3~5건 미리보기 |
| 빠른 액션 | 요청 전체 보기, 내 프로필 수정 |

## 디자인
- 다크 테마 유지
- 통계 카드: `#1a1a1a` 카드 + 레드 아이콘/숫자

## 완료 조건
- HTML 문법 오류 없음
- 반응형 (375px ~ 1440px)
